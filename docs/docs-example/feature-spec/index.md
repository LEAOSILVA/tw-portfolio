# Feature specifications

This section provides detailed specifications for selected features and components used in integration flows. Explore the pages below to understand their purpose, configuration options, and usage guidelines.

- [**Custom data format**](./data-format-custom)  
  The **Marshal** and **Unmarshal** EIPs enable interoperability between systems by transforming data between different format types. This page introduces the **Custom** format type, which allows you to process Excel spreadsheets within an integration flow using the Marshal and Unmarshal EIPs.

- [**FHIR**](./fhir)  
  The **FHIR** Apache component enables standardized healthcare data exchange between systems, ensuring interoperability and consistent communication. This page covers key aspects of using the component, including:
  - URI syntax  
  - Path and query parameters  
  - Usage examples  
  - Implementation guidelines
